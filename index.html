Ajuda

Como usar:

  1. Faça login no Waze Map Editor no seu navegador
  2. Exporte seus cookies usando uma extensão de navegador
  3. Faça upload do arquivo cookies.txt nesta aplicação
  4. Arraste os cards ou use os botões para validar places

Gestos:

  • Arrastar para direita: Marcar como lido
  • Arrastar para esquerda: Rejeitar place
  • Botões: Alternativa aos gestos

Segurança:

Seus cookies são armazenados apenas no seu navegador (sessionStorage) e enviados diretamente para as APIs do Waze através do nosso servidor proxy. Nenhum dado é salvo no servidor.

Extensões recomendadas:

  • Chrome/Edge: "Get cookies.txt LOCALLY"
  • Firefox: "cookies.txt"
