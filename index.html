Como usar:

  1. Faça login no Waze Map Editor no seu navegador
  2. Exporte seus cookies usando uma extensão de navegador
  3. Faça upload ou cole o conteúdo do cookies.txt
  4. Processe os cards (arraste ou use setas do teclado)

Importante:

Esta aplicação nunca aprova places — só rejeita ou marca como lido. Para aprovar, abra no WME pelo botão ↗ no card.

Segurança:

Os cookies são criptografados no servidor com chave secreta e descartados após 2h. O cliente só guarda um token de sessão, que pode ser invalidado a qualquer momento pelo botão de "Sair".

Extensões recomendadas:

  • Chrome/Edge: "Get cookies.txt LOCALLY"
  • Firefox: "cookies.txt"
